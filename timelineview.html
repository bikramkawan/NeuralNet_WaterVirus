Time Line View of Entire Data

Time line view is the idea to show the overview of data along the entire time frame.User can filter the entire date by providing the constraints in the table below.In order to view time line please prodived the range of each parameter. For example : If you wan to constraint heading with in the range of 100 and 300. The input will be as below.The first argument should be lowest value and second as highest value.
Example for input format of Heading = [100,360]
If you donot provide any constraint the plot will show entire data in single time frame.
Now to start the filling the arguments in the table below. Once you finished Press the Filter Button at the end of table.
ParametersInput FiltersRange Values of Dataset
Surge Velocity [m/s]Surge Ranges

Yaw Veloctiy [deg/s]Heading Ranges
